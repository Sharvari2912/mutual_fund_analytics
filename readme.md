# Mutual Fund Industry Analytics Dashboard

This project analyzes the Indian mutual fund industry using AMFI-inspired datasets from 2022–2025.

## Key Analyses
- NAV Trend Analysis
- SIP Growth Trends
- AUM Comparison
- Category Inflows
- Investor Demographics
- Geographic Distribution
- Folio Growth
- Sector Allocation
- Correlation Analysis

## Tools Used
- Python
- Pandas
- Plotly
- Seaborn
- Matplotlib
- Jupyter Notebook
